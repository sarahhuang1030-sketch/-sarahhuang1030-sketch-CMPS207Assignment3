This project builds on Assignment 2 by enhancing our Azure Web Application with:
OpenID Connect Authentication (Microsoft Entra ID)
Role-Based Authorization (OAuth2 + Claims)
Azure Key Vault for Secrets Management
Application Insights for Telemetry
Azure Monitor Alerts + Log Analytics
This is a group project divided into 5 functional roles as described below.


## Local Development
Copy `.env.example` to `.env.local` and fill in the values.
If you see an Azure SQL firewall error, ask the team lead to whitelist your IP.


## Getting Started
First, run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
